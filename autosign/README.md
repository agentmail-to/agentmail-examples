# AutoSign - AI-Powered Web Signup Automation

> **Complete web signup automation with AI-powered CAPTCHA solving and AgentMail email verification**

AutoSign is an intelligent AI agent that automates the entire signup process across any website. It uses **Browser-use** for web automation and **AgentMail** for email verification, demonstrating how to integrate both services in automated workflows.

## 🎯 What AutoSign Does

- 🤖 **AI-Powered**: Uses Claude Sonnet 4 for intelligent web automation
- 🌐 **Browser-use**: Handles web automation and form filling
- 📧 **AgentMail Integration**: Seamlessly handles email verification
- 🛡️ **CAPTCHA Solving**: Supports 2captcha integration
- 🎯 **Natural Language**: Just describe what you want in plain English
